Theme

Build the future of mobile.

Mobile API

Build an API to power the next generation of mobile apps. Recommendations, location, suggestions, photos - come up with new uses for data generated from mobile devices.

Mobile App

Can you think of the next famous mobile app or website? Use the platform you like: Web, Android, iOS or Windows. Hack your heart out and build the future.

Mobile Security

If mobile is so big, we need larger locks! Can you think of ways to enhance the security of mobile apps and APIs? Think of ways to ensure privacy in the age of ever-prevalent sharing.
